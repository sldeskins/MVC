Exercise 3: Verification

In order to verify that you have correctly performed all steps of exercise one, proceed as follows:

Verification 1

You will run the application and will create, edit and delete an address verifying that the logger is writing on the debug console when the controller's actions are invoked.

  1. To test the MVC Web Application you need to start a new instance of the MvcSampleApp project. To do this, in Solution Explorer right-click MvcSampleApp project, point to Debug and select Start New Instance.
  2. Open the Customer controller’s Index view with the list of customers. To do this, click the Customers link in the page header.
  3. Open a customer’s details page by clicking on its name.
  4. To create a new address for the selected customer, click the Add New Address link.
  5. Complete the form's input data and click on the Create button.
  6. Open the Output window on Visual Studio. To do this, go to the View menu and click Output. In the Output window, select Debug on the Show output from combo box as shown in the figure below.

    Figure 1
    Selecting the Debug pane on the Output window


  7. Verify that a log entry was added for the Create action of the AddressController controller, when it was being executed and after its execution finished.

    Figure 2
    Log entries on AddressController.Create action execution


  8. Edit the newly created address by clicking the (Edit) link, located at the right of the new address and click Save once you are on the edit form.
  9. Verify that a log entry was added for the Edit action of the AddressController controller, when it was being executed and after its execution finished.

    Figure 3
    Log entries on AddressController.Edit action execution


  10. Delete the newly created address by clicking the (Delete) link, located at the right of the new address.
  11. Verify that a log entry was added for the Delete action of the AddressController controller, when it was being executed and after its execution finished.

    Figure 4
    Log entries on AddressController.Delete action execution
