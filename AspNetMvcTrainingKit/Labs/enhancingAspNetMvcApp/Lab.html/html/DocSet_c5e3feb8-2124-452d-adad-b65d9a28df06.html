Exercise 2: Verification

In order to verify that you have correctly performed all steps of exercise one, proceed as follows:

Verification 1

You will run the application and perform a search to verify that instead of re-rendering the whole page, only the customer list is updated.

  1. To test the MVC Web Application you need to start a new instance of the MvcSampleApp project. To do this, in Solution Explorer right-click MvcSampleApp project, point to Debug and select Start New Instance.
  2. Open the Customer controller’s Index view with the list of customers. To do this, click the Customers link in the page header.
  3. Type "bike" in the Search textbox to filter the customers and click Go. The resulting screen is shown in the following figure.

    Figure 1
    The filtered customers list

    Note:
    Notice that only the customer list is updated when you click the button, and not the whole page. You can realize that by looking that the filter string still remains on the text box after submitting the form.


  4. Click the Next Page link to navigate to the next page of the customer list.
  5. Note:
    Notice that when navigating to another page only the list is updated and not the whole page, as happened on the previous step.
