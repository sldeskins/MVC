Introduction to ASP.NET MVC
Summary

In this demo, you have learnt which the core elements of an MVC application are, and how they interact. You have seen how to create an MVC application and how to use the scaffolding provided by the framework to create controllers and views. You have also seen how to unit test an ASP.NET MVC Application, specifically its routes and the controllers.
